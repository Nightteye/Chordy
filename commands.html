🎵 Music Information Commands

!artist

Music Info

Get detailed information about any artist including biography, popularity, and top tracks

Usage

!artist <artist_name>

Examples

!artist The Weeknd !artist Taylor Swift

Information Returned

  • Artist name and profile image
  • Biography and follower count
  • Genres and popularity score
  • Top 5 most popular tracks
  • Spotify verification status
  • Direct link to Spotify profile

Required Permissions

Send Messages Embed Links

!track

Music Info

Search for any track and get comprehensive details about it including audio features

Usage

!track <track_name> [artist_name]

Examples

!track Blinding Lights !track Levitating Dua Lipa

Information Returned

  • Track name and artist
  • Album information
  • Release date and duration
  • Audio features (danceability, energy, valence)
  • Explicit content warning (if applicable)
  • Streaming links (Spotify, Apple Music, etc.)

Required Permissions

Send Messages Embed Links

!album

Music Info

Get album details including full tracklist, release date, and artist information

Usage

!album <album_name> [artist_name]

Examples

!album After Hours !album folklore Taylor Swift

Information Returned

  • Album artwork and title
  • Artist name and release date
  • Total tracks and total duration
  • Complete paginated tracklist
  • Genre and record label
  • Total streams on Spotify

Required Permissions

Send Messages Embed Links

!lyrics

Music Info

Fetch song lyrics for any track with proper artist attribution and formatting

Usage

!lyrics <track_name> [artist_name]

Examples

!lyrics Bohemian Rhapsody !lyrics Imagine John Lennon

Features

  • Full lyrics in paginated format
  • Use reaction buttons (◀️ ▶️) to navigate
  • Direct link to full lyrics page
  • Artist and song credits
  • Maximum character limits for Discord

Required Permissions

Send Messages Add Reactions Manage Messages

📊 Chart Commands

!billboard

Charts

View the current Billboard Hot 100 chart with top songs and their positions

Usage

!billboard [limit]

Examples

!billboard !billboard 20 !billboard top 50

Parameters

  • limit (optional): Show top N songs (default: 10, max: 100)

Information Returned

  • Chart position and ranking number
  • Song title and artist name
  • Weeks on chart
  • Peak position achieved
  • Last week's position (↑ or ↓)

!trending

Charts

Get currently trending songs on Spotify globally or by region

Usage

!trending [region] [limit]

Examples

!trending !trending US 15 !trending IN 20

Parameters

  • region (optional): Country code (US, UK, IN, CA, AU, etc.) - default: Global
  • limit (optional): Number of tracks (default: 10, max: 50)

Information Returned

  • Trending rank position
  • Track name and artists
  • Album name
  • Current popularity score
  • Release date

!topartists

Charts

View top trending artists on Spotify by region and timeframe

Usage

!topartists [region] [limit] [timeframe]

Examples

!topartists !topartists US 20 !topartists IN 10 monthly

Parameters

  • region: Country code (default: Global)
  • limit: Number of artists (default: 10, max: 50)
  • timeframe: daily, weekly, monthly (default: weekly)

Information Returned

  • Artist rank position
  • Artist name and profile image
  • Follower count
  • Primary genres
  • Popularity score

🔍 Search & Discovery Commands

!search

Search

Search for music with advanced filters and options for tracks, artists, or albums

Usage

!search [type] <query> [filter]

Examples

!search track Blinding Lights !search artist Taylor Swift !search album folklore

Parameters

  • type: track, artist, album (default: track)
  • query: What to search for
  • filter: Optional genre or year filter

Information Returned

  • Top search results matching query
  • Results paginated for easy navigation
  • Relevance ranking
  • Quick links to streaming services

!genre

Search

Discover music by genre with personalized recommendations and curated playlists

Usage

!genre <genre_name> [limit]

Examples

!genre pop !genre jazz 15 !genre indie 20

Supported Genres

  • Pop, Rock, Hip-hop, R&B, Jazz
  • Electronic, Classical, Country
  • Latin, K-pop, Indie, Metal
  • Folk, Reggae, Blues, Soul
  • And 50+ more genres!

Information Returned

  • Popular tracks in that genre
  • Trending artists
  • Genre description
  • Related genres

📈 Statistics & Analytics Commands

!mymusic

Statistics

View your personal music statistics and preferences on the bot

Usage

!mymusic [timeframe]

Examples

!mymusic !mymusic monthly !mymusic yearly

Information Shown

  • Top 5 tracks you searched for
  • Top 5 favorite artists
  • Total searches and queries
  • Most frequently used commands
  • Favorite genres based on history

Privacy Note

  • Data is private by default
  • Only you can see your stats
  • Use !sharystats to make public

!compare

Statistics

Compare music tastes with other server members and find common interests

Usage

!compare [@user]

Examples

!compare @Alice !compare @Bob

Information Returned

  • Shared favorite artists
  • Compatibility score (0-100%)
  • Similar genres you both like
  • Cross-recommendations based on both
  • Most similar tracks

Privacy Note

  • Other user must opt-in to sharing
  • No private data shared
  • Only music preferences visible

!setprefix

Utility

Customize the command prefix for your server (server-specific)

Usage

!setprefix <new_prefix>

Examples

!setprefix ? !setprefix m! !setprefix --

Requirements

Admin Only Manage Server

Notes

  • Prefix can be 1-3 characters
  • Each server has its own prefix
  • Reset to ! with !setprefix !

Response Times

  • 100-200ms: Good

  • 500ms+: Slow

💡 Tips & Best Practices

✅ Use Exact Names

For best search results, use exact artist or track names. Partial names work but may return similar results instead of what you want.

🎯 Check Your Prefix

The default prefix is ! but server admins can customize it. Use !help to check your server's current command prefix.

⏱️ Rate Limits

Commands have reasonable rate limits to prevent abuse. If rate-limited, wait a few seconds before retrying the same command.

📄 Use Pagination

Long results use pagination. Use reaction buttons (◀️ ▶️) to navigate through pages of results in embeds.

🔒 Your Privacy Matters

Your music data is private by default. You control who sees your stats. Use !sharystats to make them public if desired.

❓ Need More Help?

Use !help for command information or join our support server. Check our documentation website for detailed guides.
